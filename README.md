# 🚗 RIDE MATE | Smart Vehicle Pooling

**RIDE MATE** is a modern, community-driven carpooling application built to make travel safer, cheaper, and more sustainable. It features a unique "Women-Only" safety mode, live GPS tracking, and a trusted review system.

## ✨ Key Features
* **📍 Live Route Tracking:** Real-time distance and ETA calculations using Leaflet.js and OpenStreetMap.
* **⭐ Passenger Reviews:** Integrated 5-star rating system to build community trust.
* **🌸 Women-Only Mode:** Dynamic UI themes (Pink/Sage) and filtered ride options for female safety.
* **💳 Payment Management:** Seamless tracking of trip costs and ride history.
* **📱 Glassmorphic Design:** A beautiful, responsive interface with a modern frosted-glass aesthetic.

## 🛠️ Tech Stack
- **Backend:** Python (Flask)
- **Database:** SQLAlchemy (SQLite)
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript
- **Maps API:** Leaflet.js & Nominatim
